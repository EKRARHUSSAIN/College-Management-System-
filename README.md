MECW – Mewat Engineering College Management System

Live Demo: https://mecw.netlify.app/

A comprehensive college management system designed to streamline administration, enhance learning, and foster growth for modern educational institutions.

✨ Key Features

· Student Management – Digital admissions, attendance tracking, performance analytics
· Faculty Portal – Course management, digital gradebook, schedule planning
· Administration – Fee management, library system, transport & hostel
· Smart Scheduling – Automated timetable management
· Role-based Access – Separate dashboards for Admin, Faculty, Students, and Parents

🛠️ Tech Stack

· Frontend: React.js
· Backend: Node.js / Django (specify as per your project)
· Database: MySQL / PostgreSQL
